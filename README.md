## 🎯 Visão Geral
Sistema de cadastro e gerenciamento de tarefas em colunas **A Fazer → Fazendo → Pronto** com integração total.<br><br>


## ⚙️ Execução do sistema

### 1️⃣ Criação do banco de dados + tabela

Execute o código SQL localizado em:

```
backend/database/setup_banco.sql
```
### 2️⃣ Instalar NPM, Express e Pg

Instale o Node.js e depois no terminal, na pasta raiz do projeto, digite o comando:

```
npm install
```

### 3️⃣ Execução do sistema

Na pasta raiz do projeto, coloque o seguinte comando:

```
node app.js
```

### 4️⃣ Visualização do site

Na barra de pesquisa do navegador, coloque:

```
http://localhost:3000
```
