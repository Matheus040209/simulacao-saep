## 🎯 Visão Geral
Sistema de cadastro e gerenciamento de tarefas em colunas **A Fazer → Fazendo → Pronto** com integração total.


## ⚙️ Instalação Rápida

### 1️⃣ Backend + Banco
```bash
cd backend
npm install
# Execute setup_banco.sql no PostgreSQL
npm run dev  # http://localhost:3000
```

### 2️⃣ Frontend
```
Backend rodando → Abrir http://localhost:3000/public/index.html
```

### Comandos
```bash
npm run dev     # Dev (nodemon)
npm start       # Prod
```
